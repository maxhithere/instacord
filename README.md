Contributing

# Step 1:
- Fork this repo
# Step 2:
- Install all dependecies with `npm install`
# Step 3:
- Create a `.env` file in your main path, then add all necessary vars as seen in `.env.example`
# Step 4:
- Make any necessary changes
# Step 5:
- Submit a pull request
